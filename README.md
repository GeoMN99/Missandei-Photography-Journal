# Through The Lens — A Photography Portfolio

# Author
Missandei Narth



# Description
Through The Lens is a photography portfolio website that celebrates the art of visual storytelling. The site features a curated gallery of favourite photographs, a biography of the photographer, and a contact form for inquiries.



# Features
- Fixed navigation header with smooth scroll links
- Full screen hero section with background image
- Introductory paragraph about the website
- Photo gallery with favourite photos
- Photographer biography with portrait image
- Contact form with validation
- Custom CSS stylesheet
